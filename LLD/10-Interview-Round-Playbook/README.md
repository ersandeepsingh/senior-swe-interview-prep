# Interview Round Playbook — How to Run an LLD Round

Senior machine-coding is a **process**, not a dump of patterns. This folder expands the five-step playbook from [LLD-Patterns-Senior-SWE.md](../LLD-Patterns-Senior-SWE.md) into practical guides you can rehearse out loud.

**Shared domain:** every guide continues the same **Parking Lot** problem so phrases, entities, and code stay coherent across steps.

| Step | Time | File |
|------|------|------|
| 1. Clarify & scope | ~5 min | [01_clarify_and_scope.md](01_clarify_and_scope.md) |
| 2. Identify entities & relationships | ~10 min | [02_identify_entities_and_relationships.md](02_identify_entities_and_relationships.md) |
| 3. Pick load-bearing patterns | ~5 min | [03_pick_load_bearing_patterns.md](03_pick_load_bearing_patterns.md) |
| 4. Code skeleton / end-to-end flow | ~25–35 min | [04_code_skeleton_end_to_end.md](04_code_skeleton_end_to_end.md) |
| 5. Concurrency, extensibility, testing | ~5–10 min | [05_concurrency_extensibility_testing.md](05_concurrency_extensibility_testing.md) |
| Mock rubric (full round) | 45–60 min | [06_mock_interview_rubric.md](06_mock_interview_rubric.md) |

**How to drill:** pick Parking Lot (or Checkout) → run steps 1–5 timed → grade yourself with [06](06_mock_interview_rubric.md). Say the phrases out loud; silence kills senior signal.
